# revealjs-skill for Claude Code

## Summary
Claude Code用のreveal.jsスキル。自然言語プロンプトからプロフェッショナルなHTML形式のプレゼンテーションを生成する。ビルドステップ不要。

## Key Points
- Plugin方式でインストール可能（`/plugin marketplace add ryanbbrown/revealjs-skill`）
- Playwright によるオーバーフロー検出
- DeckTape によるPDF出力
- カスタムCSS、マルチカラム、Chart.js対応
- ブラウザ上でのインライン編集機能あり

## Relevance
Claude Codeからスライドを生成する主要な方法の一つ。Webプレゼンテーション向き。
