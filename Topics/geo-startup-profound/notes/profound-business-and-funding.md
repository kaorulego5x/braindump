# Profoundのビジネスモデルと資金調達

## Summary

ProfoundはSaaS型のGEOプラットフォームで、創業からわずか18ヶ月でユニコーン企業（評価額10億ドル）に到達。累計1億5,500万ドル以上を調達し、700以上のエンタープライズ顧客（Fortune 500の10%含む）を獲得。Lightspeed、Sequoia Capital、Kleiner PerkinsなどトップティアのVCが支援する。

## Notes

### 資金調達の経緯

| ラウンド     | 金額         | リード投資家                      | 時期         |
| -------- | ---------- | --------------------------- | ---------- |
| Seed     | $3.5M      | -                           | 2024年      |
| Series A | $20M       | Kleiner Perkins             | 2025年初頭    |
| Series B | $35M       | Sequoia Capital             | 2025年8月    |
| Series C | $96M       | Lightspeed Venture Partners | 2026年2月24日 |
| **合計**   | **$155M+** |                             |            |

Series Cにおける評価額は**10億ドル（$1B）**で、創業から約18ヶ月でユニコーンステータスに到達。他の参加投資家にはSaga VC、South Park Commons、Evantic Capitalが含まれる。

### 創業者・経営チーム

- **James Cadwallader**（共同創業者・CEO）: 英国出身のシリアルマーケティング起業家
- **Dylan Babbs**（共同創業者・CTO）: 元Uberソフトウェアエンジニア、ワシントン大学出身

二人はサンフランシスコのSouth Park Commonsで出会った。South Park CommonsはFacebook初の女性エンジニアRuchi Sanghviと元Dropbox CTOのAditya Agarwalが2016年に設立したコミュニティ兼初期ステージファンド。

### ビジネスモデル

- **SaaS（サブスクリプション）**: 月額$99〜カスタムエンタープライズ価格
- **ターゲット顧客**: 大手エンタープライズブランド（マーケティングチーム向け）
- **セルフサーブ + エンタープライズセールス**: Liteプランでスタートアップ・代理店も対象

### 主要顧客

Target、Walmart、Figma、Ramp、MongoDB、U.S. Bank、Chime、Indeed、DocuSign、Clay、Plaid

- **700+**エンタープライズ顧客
- **Fortune 500の10%**が利用

### 成長速度

Profoundの成長速度は特筆すべきもので、Sequoiaが「マーケティングの次の時代のSalesforceになる」と評した。創業から約1年半で10億ドル評価額に到達する速度は、SaaSスタートアップとしても異例。

### Series Cの使途

- AIドリブンのマーケティングエージェント（Profound Agents）の開発拡大
- ロンドンオフィス開設（グローバル展開）
- 「エージェント型マーケティングインフラ」の構築
